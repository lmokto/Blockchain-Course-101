Here, we try to do the actual mining
We use a proof-of-work mechanism
It works by taking a block and hashing it :-
  1. We use the SHA256 algorithm
  2. We need to get the hash as a string
  3. We check if this hash string has 4 zeros in the beginning
  4. if conditions are met, it is a valid block
  5. if not, we increment the nonce and try again, till infinity and beyond!
