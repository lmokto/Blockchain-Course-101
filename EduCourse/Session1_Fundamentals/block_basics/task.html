Let's see how a simple block looks - pythonically
A block usually consists of the following :-
  • Some data : In bitcoin, this is a message you can write along with your transaction
    For example : "Home rent" , "Pizza"
  • Hash of the previous block This is very important. This is what connects the blocks and says which one is after which one. The only exception is the genesis block, i.e the first block
  • TimeStamp This allows you to decide which block was mined earlier. If two blocks were mined together, the one that was mined earlier is accepted
  • Other Data Which include the index, nonce, and other stuff. We will come back to this later
